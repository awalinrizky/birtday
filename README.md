# Our Little Place

A premium, long-term digital memory platform and relationship timeline. 
Initially built as a birthday gift, architected as a production-ready SaaS application.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express.js
- **Database**: Supabase (PostgreSQL)
- **Storage**: Supabase Storage
- **Deployment**: Vercel (Frontend), Railway (Backend)

## Architecture
This repository is split into two main directories:
- `/frontend` - Client-facing application and Admin Dashboard.
- `/backend` - RESTful API and authentication services.
